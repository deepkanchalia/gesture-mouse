Point, open, drag — through your webcam. No hardware, no mouse. Free and open source.

macOS Open source Runs 100% on your Mac — nothing leaves it

Install — 3 steps

1

Paste one line in Terminal

Open Terminal (press ⌘ + Space, type "Terminal") and paste:

curl -fsSL https://raw.githubusercontent.com/deepkanchalia/gesture-mouse/main/install.sh | bash

It downloads the app and sets everything up. Needs Python 3.11 — if you don't have it, the installer tells you the exact command to get it.

2

Allow camera + mouse control

System Settings → Privacy & Security → enable Camera and Accessibility for Terminal. Without Accessibility, the cursor won't move.

3

Run it

cd ~/gesture-mouse && ./venv/bin/python main.py

A camera window opens with live coaching text. Raise your hand — the cursor follows.

The gestures

👋
Move your hand — cursor follows
🎯
Hold still 2s on a target — locks on (armed). Nothing ever clicks without this.
🤏
Flick finger + thumb 3× — opens it (double-click)
✊
Close your hand 2s — picks it up · open your hand to drop
🛑
Hand out of frame — cursor freezes. ESC quits.
Privacy: the camera feed never leaves your Mac. No internet, no accounts, no data collected. It's ~600 lines of Python you can read yourself.
