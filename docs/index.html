Install — 3 steps

1

Paste one line in Terminal

Open Terminal (press ⌘ + Space, type "Terminal") and paste:

curl -fsSL https://raw.githubusercontent.com/deepkanchalia/gesture-mouse/main/install.sh | bash

It downloads the app and sets everything up. Needs Python 3.11 — if you don't have it, the installer tells you the exact command to get it.

2

Allow camera + mouse control

System Settings → Privacy & Security → enable Camera and Accessibility for Terminal. Without Accessibility, the cursor won't move.

3

Run it

cd ~/gesture-mouse && ./venv/bin/python main.py

A camera window opens with live coaching text. Raise your hand — the cursor follows.

The gestures

👋
Move your hand — cursor follows
🎯
Hold still 2s on a target — locks on (armed). Nothing ever clicks without this.
🤏
Flick finger + thumb 3× — opens it (double-click)
✊
Close your hand 2s — picks it up · open your hand to drop
🛑
Hand out of frame — cursor freezes. ESC quits.
Privacy: the camera feed never leaves your Mac. No internet, no accounts, no data collected. It's ~600 lines of Python you can read yourself.

Troubleshooting

The cursor doesn't move

That's the Accessibility permission. Go to System Settings → Privacy & Security → Accessibility, add your Terminal app and switch it on, then quit and re-run. macOS blocks mouse control silently until this is granted.

The camera shows my iPhone instead of my webcam

macOS Continuity Camera hijacked the feed (you may see "Lock iPhone To Resume"). Fix: on your iPhone, Settings → General → AirPlay & Continuity → Continuity Camera OFF. Re-run the app.

"Python 3.11 or 3.12 not found"

The hand-tracking library (MediaPipe) doesn't support newer Pythons yet. Install it with brew install python@3.11, then run the install command again. No Homebrew? Get it at brew.sh first.

Is this safe to run?

Yes. It runs 100% locally — there is zero network code in the app, no accounts, no analytics. It's MIT-licensed open source, small enough to read in one sitting: read the source.

How do I stop it?

Three ways: press ESC in the camera window, take your hand out of the frame (cursor freezes instantly), or just close the camera window. Nothing gets clicked without the 2-second arm step, so it can't run away from you.

Gestures misfire on my hand

Every hand is different. Run ./venv/bin/python pinch_debug.py — it logs your hand's real numbers to pinch_log.txt so you can tune the thresholds in main.py to data, not guesswork. Tuning knobs are documented in the README.
